שכבות (סדר בנייה)

  1. ✓ הושלם · Debate Engine (core) — Hegelian flow, Aya 32B, מערכת system prompt דו-לשונית
  2. ✓ הושלם · Persistent Memory — SQLite + סכמה (debates/turns/synthesis_moments/lessons/knowledge_chunks)
  3. ✓ הושלם · RAG Retrieval — sqlite-vec, embeddings, formatContextBlock (תוקן באג BigInt — embedding inserts נכשלו בשקט)
  4. ✓ הושלם · API + UI integration — sidebar עם היסטוריה, ויכוח חדש, טעינת ויכוח
  5. ✓ הושלם · Self-Critique Loop — POST /api/critique/[id], JSON-rubric, lesson נשמר ל-vec_lessons
  6. ממתין · Strategy Evolution — Elo per topic, אסטרטגיות שמשתפרות
  7. ✓ הושלם · Knowledge Ingestion — POST /api/ingest, scripts/ingest.py, chunker חכם, פופר טען לבחינה
  8. ממתין · Autonomous Self-Practice — AI מתאמן נגד עצמו ברקע (היעד הגבוה ביותר)

אומת: שאילתת "פופר הפרכה מדע" מחזירה 5 chunks מהמסמך + 2 turns + 1 lesson, ממויינים לפי distance. RAG חי באמת עכשיו.

חשוב: באג קריטי תוקן — node:sqlite חיבר JS numbers כ-REAL ל-vec0, מה שגרם לכל insert של embedding להיכשל בשקט מאז ההתחלה. נדרש BigInt(id). שווה תיעוד ב-memory.
